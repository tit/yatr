<h1>YaTr</h1>

<h2>Description</h2>
Extended String Class for translate with API Yandex

<h2>Usage</h2>
<pre>
require "yatr"

puts "hello".to_ru # => привет
puts "hello".to_en # => hello
puts "hello".to_pl # => hello
puts "hello".to_uk # => привiт
puts "hello".to_de # => hallo
puts "hello".to_tr # => merhaba
</pre>
